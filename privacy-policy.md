Privacy Policy for GitHub Productivity Extension

Last Updated: March 2026

GitHub Productivity Extension does not collect, store, transmit, or share any personal data.

All functionality of this extension runs locally in the user's browser. Any settings or temporary data used by the extension are stored locally using the browser's storage APIs (such as localStorage or sessionStorage) and are never transmitted to external servers.

This extension does not use analytics, tracking scripts, or third-party services.

Permissions requested by the extension are used solely to enable functionality on GitHub pages.
